Installation

To install FutureSonic, double-click the FutureSonic Installer and follow the on-screen instructions. The Installer will tell you when the installation process is complete. FutureSonic is installed as a Windows service, and is started automatically when the computer starts. Double-click the FutureSonic icon in the Windows System Tray to launch the FutureSonic control panel.

By default, FutureSonic is accessible on http://localhost:4040/, but the port number and URL can be changed from the control panel.

If you are upgrading an existing installation of FutureSonic, your previous settings (stored in c:\futuresonic) will be preserved.
